`OpenProxy` is a self-hosted AI API gateway that provides a unified interface, multi-provider routing, authentication, billing, and an admin dashboard.

## Who Is This For

- Developers who want to run the full stack locally (Rust Proxy + Bun Server + Web)
- Teams that need self-hosted deployment with their own database and auth/payment integrations
- Applications that want a unified OpenAI/Anthropic-style interface for multiple providers

## Project Components

- `apps/api`: Rust + Axum proxy layer for request forwarding and usage tracking
- `apps/server`: Bun + Elysia backend for auth, management, billing, and configuration
- `apps/web`: React + Vite frontend (tenant panel + admin panel)
- `packages/*`: Shared packages (schema, payment, phone auth, etc.)
